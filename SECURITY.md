# Security Policy

RelayProbe reports observable anomalies in user-authorized relay tests. Please do not submit real secrets in bug reports.

## Reporting

Open a private security advisory or contact the maintainer before publishing a vulnerability that includes:

- real credentials
- personal data
- private relay logs
- reproducible cross-user leakage evidence

## Scope

Security issues in RelayProbe itself are in scope. Claims about a third-party relay require permission from that relay operator and should be handled responsibly.
